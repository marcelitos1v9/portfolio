Termocity

Este é um projeto web chamado "Termocity" que fornece informações climáticas precisas e atualizadas para qualquer cidade do Brasil. O usuário insere o nome da cidade no campo de busca e clica no botão "Obter Temperatura". A temperatura atual, condições climáticas, umidade do ar, velocidade do vento e outras informações úteis são exibidas para o usuário em tempo real. Além disso, o site oferece informações adicionais, como previsão do tempo, para quem deseja se manter informado sobre as condições climáticas da cidade. O projeto utiliza as tecnologias HTML, CSS, Bootstrap e JavaScript para criar uma interface de usuário atraente e responsiva.

Pokedex

A Pokedex é um aplicativo web desenvolvido com HTML, CSS e JavaScript, que utiliza a API pública do PokeAPI para obter informações sobre os Pokemons. Com a Pokedex, é possível pesquisar informações sobre um Pokemon por nome ou ID, exibindo detalhes como nome, ID, tipo, altura e peso. A interface do usuário é simples e intuitiva, contando com um campo de busca e botões para navegar pelos Pokemons. A Pokedex é responsiva, podendo ser utilizada em dispositivos móveis e desktops.

Tecnologias front-end

Html

O HTML é uma linguagem de marcação que é usada para criar conteúdo estruturado e semântico na web. HTML significa "Hypertext Markup Language" (Linguagem de Marcação de Hipertexto) e é a base de praticamente todos os sites da internet.

CSS

O CSS, ou "Cascading Style Sheets" (Folhas de Estilo em Cascata), é uma linguagem de estilo que é usada em conjunto com o HTML para adicionar estilo e formatação a uma página web. O CSS permite que os desenvolvedores especifiquem como o conteúdo HTML deve ser apresentado na página, controlando coisas como cores, fontes, layout e muito mais..

Javascript

JavaScript é uma linguagem de programação de alto nível, interpretada e orientada a objetos. É frequentemente usada para criar páginas da web interativas e dinâmicas, além de aplicativos web e móveis. JavaScript é executado nos navegadores dos usuários e permite a manipulação do conteúdo da página em tempo real, interação com o usuário e validação de formulários.

PHP

PHP (Hypertext Preprocessor) é uma linguagem de programação de código aberto especialmente adequada para o desenvolvimento de aplicativos da web. Com PHP, é possível criar sites dinâmicos e interativos, gerar conteúdo dinamicamente, gerenciar bancos de dados e manipular arquivos. Ele é muito popular entre os desenvolvedores devido à sua simplicidade, flexibilidade e facilidade de uso, além de ser amplamente suportado pela maioria dos sistemas operacionais e servidores da web.

SQL

SQL (Structured Query Language) é uma linguagem de programação usada para gerenciar e manipular dados em um banco de dados relacional. Ela é usada para criar, modificar e consultar dados em bancos de dados relacionais, que são usados para armazenar informações em tabelas com relações entre si.
